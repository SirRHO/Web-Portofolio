PENGEMBANGAN MENU EDUKASI & KARIR PADA WEBSITE CV.NATUSI

Internship Project • CV.Natusi • Web Application

Deskripsi

Project ini mengembangkan fitur Edukasi dan Karir pada website CV.Natusi untuk mempermudah pengelolaan informasi lowongan kerja, magang, pelatihan, dan kursus, dengan tampilan responsif di desktop maupun perangkat android.

Tools & Teknologi

Figma VS Code Bootstrap Laravel JavaScript PHP CSS MySQL

Jobdesk

  • Merancang dan membangun fitur Edukasi dan Karir pada website CV.Natusi berbasis Laravel
  • Merancang tampilan responsif agar nyaman digunakan di desktop maupun perangkat android.
  • Melakukan pengujian dan penyesuaian sistem sesuai kebutuhan perusahaan untuk memastikan fungsionalitas berjalan dengan baik.
