PENGEMBANGAN APLIKASI IOT PENYIRAMAN OTOMATIS DAN MONITORING TANAMAN BAWANG PREI BERBASIS FLUTTER

Pengabdian Masyarakat • Desa Kadangan • Android Application

Deskripsi

Project ini mengembangkan aplikasi Flutter berbasis IoT untuk monitoring dan penyiraman otomatis tanaman bawang prei secara real-time.

Jobdesk

  • Mendesain UI/UX aplikasi mobile dengan Flutter, memastikan tampilan responsif dan mudah digunakan.
  • Merancang dan membangun aplikasi mobile menggunakan Flutter, menghadirkan fungsionalitas sesuai kebutuhan pengguna.
  • Mengintegrasikan Firebase untuk backend dan database, mendukung penyimpanan data dan interaksi real-time.

Tools & Teknologi

Figma VS Code Flutter Dart Firebase
